1. Lexical.py

(1) 주어진 code를 token화 하는 작업

(2) lexical : code를 분리하여 token 저장하도록 한다

(3) get_digit : 첫 값이 num인 경우, 계속해서 num이 나오지 않을 때까지 추가적인 input 확인작업을 한다.
		ex) 12 + 13 == 25 --> token : [12]

(4) get_str : 첫 값이 char인 경우, 계속해서 char이 나오지 않을 때까지 추가적인 input 확인 작업을 한다.
		ex) abcd + a == 12 --> token : [abcd]

(5) 나머지는 if문을 사용하여 분류

(6) int, char은 소문자 if, then, else, while,exit은 대문자로 작성하였음 유의 바람

(7) space bar와 enter key도 scanner에 반영

(8) 적어도 하나라도 error가 나올 시 해당 scanner를 종료한다.


2. Lexical_check.txt

(1) 해당 text를 Starting.py에 넣어 확인한다.

(2) 텍스트 파일에 금지 expression ex) "#" 등을 사용할 수 없다
